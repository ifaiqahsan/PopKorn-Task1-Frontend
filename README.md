# PopKorn - Movie Ticket Booking System

## Overview
PopKorn is a responsive movie ticket booking platform developed as part of the DecodeLabs Summer Internship Program 2026.

## Features
- Browse Movies
- Search & Filter Movies
- View Movie Details
- Seat Selection
- Booking History
- Responsive Design

## Technologies
- React
- Vite
- JavaScript
- HTML5
- CSS3

## Installation

npm install
npm run dev
