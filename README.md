# 🍿 PopKorn - Movie Ticket Booking System

A high-fidelity, responsive frontend React application for browsing cinema titles, applying advanced genre filters, and managing active electronic booking passes.

## 📱 Interface Preview & Screenshots

### 🎬 Explore Movies Catalog
Browse real-time cinema titles with interactive filling gauges, advanced live search query filters, and clean genre sorting layouts.
