# 🍿 PopKorn - Movie Ticket Booking System

A high-performance, responsive React application built to provide a seamless cinema experience. From browsing trending titles to verifying electronic booking passes, PopKorn demonstrates modern frontend architecture and interactive UI/UX patterns.

---

## 🚀 Key Features

* **Advanced Catalog Browser:** Dynamic filtering by genre, runtime, and language.
* **Interactive Seating Engine:** Randomized seat availability generator with real-time price calculation logic.
* **Electronic Pass System:** Verified booking passes with unique reference identifiers and simulated tracking barcodes.
* **Premium UI/UX:** Glassmorphism-inspired design system with high-contrast, cinema-themed aesthetics.
